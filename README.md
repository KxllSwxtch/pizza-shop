## Getting Started

1. Clone the project

```
git clone https://github.com/pomegranitedesign/pizza-shop.git
```

2. Install packages

```py
npm i
or
yarn
```

3. Start project

```py
npm run start
or
yarn start
```

## Additional notes

- To build the project run

```py
npm run build

yarn build
```
